# TrioZ — Экосистема проектов

Веб-экосистема проектов T.Р.И.О.Z с dark fantasy эстетикой и Apple/Google стилем анимаций.

## Разделы

- **Проекты Т.Р.И.О."Z"** — MMORPG, игры, карты, онлайн
- **Перо Измерений** — книги, настольные игры, офлайн
- **TZ.Connect** — мессенджер + голосовая связь + IT-услуги для бизнеса
- **TZ.Library** — база знаний, лор, вики

## Стек технологий

- **Next.js 14** (App Router) + TypeScript
- **Tailwind CSS** + **Framer Motion**
- **Prisma** + SQLite
- **NextAuth.js** (аутентификация)

## Запуск

```bash
npm install
npx prisma migrate dev
npm run seed
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Демо-доступ

- **Админ**: admin@trioz.ru / admin123
- **Пользователь**: user@trioz.ru / user123

## Админ-панель

Доступна по `/admin` для пользователей с ролью ADMIN:
- Управление пользователями (баны, роли, удаление)
- Управление контентом (статьи, категории)
- Управление услугами
- Управление элементами экосистемы
